HTML Guide


*HTML Headings

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Output

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

*Paragraph Element

<p>This is a Paragraph</p>

Output:

This is a Paragraph

*Line break element

<p>text is here<br>
A new text is here</p>

Output

Text is here
A new text is here

*HTML Lists

*Ordered list

<ol>
<li> coffee</li>
<li>Tea</li>
<li>Juice</li>
</ol>

Output

  1. coffee
  2. Tea
  3. Juice

*Unordered list

<ul>
<li> Coffee</li>
<li>Tea</li>
<li>Juice</li>
</ul>

Output


  • Coffee
  • Tea
  • Juice
